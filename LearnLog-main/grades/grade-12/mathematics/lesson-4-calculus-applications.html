Introduction to Calculus Applications

This lesson synthesizes all calculus techniques to solve complex real-world problems. From optimizing business profits to modeling physical systems, calculus provides the tools to analyze change, find extrema, and understand the behavior of functions. We'll explore optimization, curve analysis, physics applications, and economic modeling.

1. Optimization Problems

Optimization finds maximum or minimum values subject to constraints, critical in engineering, economics, and design.

Strategy for Optimization

  1. Identify the quantity to optimize (objective function)
  2. Express it as a function of one variable using constraints
  3. Find critical points: set f'(x) = 0
  4. Use first or second derivative test to classify extrema
  5. Check endpoints if on a closed interval

Example 1: Maximizing Area

A farmer has 400 meters of fence to enclose a rectangular pasture along a river (one side needs no fence). What dimensions maximize the area?

Solution:

Let x = width (parallel to river), y = depth

Constraint: x + 2y = 400 (total fence)

Area: A = xy = x(200 - x/2) = 200x - x²/2

Find maximum: A'(x) = 200 - x = 0 → x = 200 meters

Then y = (400 - 200)/2 = 100 meters

Second derivative: A''(x) = -1 < 0 (confirms maximum)

Maximum area: A = 200(100) = 20,000 m²

Example 2: Minimizing Cost

A cylindrical can must hold 1000 cm³. Find dimensions that minimize the material (surface area).

Solution:

Volume: V = πr²h = 1000 → h = 1000/(πr²)

Surface area: S = 2πr² + 2πrh (top + bottom + side)

Substitute h: S(r) = 2πr² + 2πr(1000/(πr²)) = 2πr² + 2000/r

Find minimum: S'(r) = 4πr - 2000/r² = 0

4πr³ = 2000 → r³ = 500/π → r ≈ 5.42 cm

h = 1000/(π(5.42)²) ≈ 10.84 cm

Notice: h = 2r (optimal cylinder has height twice the radius)

2. Curve Sketching and Analysis

Use derivatives to understand function behavior completely.

Complete Analysis Checklist

  1. Domain: Where is f(x) defined?
  2. Intercepts: x-intercepts (f(x) = 0), y-intercept (f(0))
  3. Symmetry: Even (f(-x) = f(x)), odd (f(-x) = -f(x))
  4. Asymptotes: Vertical (undefined points), horizontal (limits at ±∞)
  5. First derivative f'(x): Critical points, increasing/decreasing intervals
  6. Second derivative f''(x): Inflection points, concavity

Example: Analyze f(x) = x³ - 6x² + 9x + 1

Analysis:

Domain: All real numbers

f'(x) = 3x² - 12x + 9 = 3(x - 1)(x - 3)

Critical points: x = 1, x = 3

f''(x) = 6x - 12 = 6(x - 2)

Inflection point: x = 2, f(2) = 3

Intervals:

Increasing: x < 1 or x > 3 (f'(x) > 0)

Decreasing: 1 < x < 3 (f'(x) < 0)

Concave down: x < 2 (f''(x) < 0)

Concave up: x > 2 (f''(x) > 0)

Local max at x = 1: f(1) = 5

Local min at x = 3: f(3) = 1

3. Physics Applications

Projectile Motion

An object launched at angle θ with initial velocity v₀:

  • Horizontal: x(t) = v₀cos(θ)·t
  • Vertical: y(t) = v₀sin(θ)·t - (1/2)gt²

Example: Optimal Launch Angle

Find the angle that maximizes range for projectile with v₀ = 30 m/s (g = 9.8 m/s²).

Solution:

Range R = (v₀²sin(2θ))/g

R'(θ) = (2v₀²cos(2θ))/g = 0

cos(2θ) = 0 → 2θ = 90° → θ = 45°

Maximum range: R = (30²)/9.8 ≈ 91.8 meters

Work and Energy

Work done by variable force F(x) from a to b:

W = ∫ₐᵇ F(x) dx

Example: Pumping Water from Tank

A cylindrical tank (radius 2 m, height 6 m) is full of water. How much work to pump all water to the top?

Solution:

Consider horizontal slice at depth y (measured from bottom):

Volume of slice: dV = πr²dy = π(2²)dy = 4π dy

Mass: dm = ρ·dV = 1000(4π dy) kg (ρ_water = 1000 kg/m³)

Distance to lift: (6 - y) meters

Work for slice: dW = (6 - y)·1000·4π·9.8 dy

Total work: W = ∫₀⁶ 39200π(6 - y) dy

= 39200π[6y - y²/2]₀⁶

= 39200π(36 - 18) = 705,600π ≈ 2,217,000 Joules

4. Economic Applications

Profit Maximization

Profit: P(x) = R(x) - C(x)

Maximize when: P'(x) = 0 → R'(x) = C'(x)

Marginal revenue equals marginal cost

Example: Manufacturing Optimization

A company has cost C(x) = 5000 + 30x + 0.02x² and revenue R(x) = 80x - 0.01x² for x units.

Find optimal production:

Profit: P(x) = R(x) - C(x) = -5000 + 50x - 0.03x²

P'(x) = 50 - 0.06x = 0 → x ≈ 833 units

P''(x) = -0.06 < 0 (confirms maximum)

Maximum profit: P(833) ≈ $15,833

Elasticity of Demand

Measures price sensitivity:

E(p) = -(p/q)·(dq/dp)

  • E > 1: Elastic (quantity very sensitive to price)
  • E < 1: Inelastic (quantity less sensitive)
  • E = 1: Unit elastic (revenue maximized)

Example: Demand Analysis

Demand: q = 1000 - 5p. Find elasticity at p = $80.

Solution:

dq/dp = -5

At p = 80: q = 1000 - 5(80) = 600

E(80) = -(80/600)(-5) = 400/600 ≈ 0.67

Since E < 1, demand is inelastic at this price

A price increase would increase total revenue

5. Related Rates in Complex Systems

Example: Conical Tank Drainage

A conical tank (vertex down, radius 3 m, height 6 m) drains at 2 m³/min. How fast is the water level dropping when depth is 4 m?

Solution:

Volume of cone: V = (1/3)πr²h

Similar triangles: r/h = 3/6 = 1/2 → r = h/2

V = (1/3)π(h/2)²h = πh³/12

Differentiate: dV/dt = (πh²/4)(dh/dt)

Given: dV/dt = -2 m³/min, h = 4 m

-2 = (π·16/4)(dh/dt)

dh/dt = -2/(4π) = -1/(2π) ≈ -0.159 m/min

Water level drops at about 15.9 cm/min

6. Volume and Area Applications

Solids of Revolution

Rotating region around x-axis:

Volume = π∫ₐᵇ [f(x)]² dx (disk method)

Example: Paraboloid Volume

Find volume when y = x² from x = 0 to x = 2 is rotated around the x-axis.

Solution:

V = π∫₀² (x²)² dx = π∫₀² x⁴ dx

= π[x⁵/5]₀² = π(32/5) = 32π/5 ≈ 20.1 cubic units

Practice Problems

Problem 1: Optimization

A rectangular box with square base must have volume 32 m³. Find dimensions that minimize surface area.

Problem 2: Curve Sketching

Completely analyze and sketch f(x) = x/(x² + 1), including asymptotes, extrema, and inflection points.

Problem 3: Physics

A ball is thrown upward from ground level at 25 m/s. When does it reach maximum height? What is the maximum height?

Problem 4: Economics

Cost C(x) = 1000 + 20x + 0.05x², Revenue R(x) = 100x - 0.1x². Find production level that maximizes profit and calculate the maximum profit.

Problem 5: Related Rates

A spherical balloon is inflated at 50 cm³/s. How fast is the surface area increasing when radius is 10 cm?

Problem 6: Volume

Find the volume when the region between y = sin(x) and y = 0 from x = 0 to x = π is rotated around the x-axis.

Key Takeaways

  • Optimization: Find extrema by setting f'(x) = 0 and testing with second derivative
  • Complete curve analysis uses domain, intercepts, asymptotes, f'(x), and f''(x)
  • Physics applications: projectile motion, work (∫F dx), pumping problems
  • Economics: maximize profit when marginal revenue equals marginal cost
  • Elasticity measures price sensitivity of demand
  • Related rates connect rates of change of multiple variables
  • Volumes of revolution use disk method: V = π∫[f(x)]² dx
  • Always verify critical points are maxima/minima, not just inflection points
  • Real-world problems require translating constraints into mathematical expressions
