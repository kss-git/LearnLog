The Reverse of Differentiation and Area Under Curves

← Back to Mathematics

What is Integration?

Integration is the reverse process of differentiation. While derivatives measure rates of change, integrals measure accumulation - total distance from velocity, total area under a curve, or total quantity from a rate.

Two main types:

  • Indefinite Integral: ∫f(x)dx - finds the antiderivative (reverse of derivative)
  • Definite Integral: ∫abf(x)dx - finds the area under the curve from a to b

📊 Area Under a Curve

The definite integral represents the signed area between the curve and the x-axis.

🔄 The Fundamental Theorem of Calculus

This theorem connects derivatives and integrals - they are inverse operations!

If F'(x) = f(x), then ∫ᵇₐf(x)dx = F(b) - F(a)

In plain English: To find the area under f(x), find its antiderivative F(x), then subtract F(a) from F(b).

📐 Power Rule for Integration

Just like derivatives have a power rule, so do integrals:

∫xⁿdx = (xⁿ⁺¹)/(n+1) + C

(where n ≠ -1, and C is the constant of integration)

Example 1: ∫x⁴dx = x⁵/5 + C

Example 2: ∫(3x² + 2x - 5)dx = x³ + x² - 5x + C

Example 3: ∫₁³x²dx = [x³/3]₁³ = 27/3 - 1/3 = 26/3

🚗 Real-World: Distance from Velocity

A car's velocity is v(t) = 2t + 3 m/s. How far does it travel between t = 0 and t = 5 seconds?

Solution:

Distance = ∫₀⁵(2t + 3)dt

= [t² + 3t]₀⁵

= (25 + 15) - (0 + 0)

= 40 meters

📈 Area Between Curves

To find the area between two curves f(x) and g(x) where f(x) ≥ g(x):

Area = ∫ᵇₐ[f(x) - g(x)]dx

🎯 Practice Problems

Problem 1: Indefinite Integral

Find: ∫(x³ - 4x + 7)dx

Show Solution

Apply power rule to each term:

∫(x³ - 4x + 7)dx = x⁴/4 - 2x² + 7x + C

Problem 2: Definite Integral

Evaluate: ∫₁⁴(2x - 1)dx

Show Solution

∫₁⁴(2x - 1)dx = [x² - x]₁⁴

= (16 - 4) - (1 - 1)

= 12 - 0 = 12

Problem 3: Area Under Curve

Find the area under y = x² from x = 0 to x = 3.

Show Solution

Area = ∫₀³x²dx = [x³/3]₀³

= 27/3 - 0 = 9 square units

Problem 4: Total Distance

A particle's velocity is v(t) = 6t² - 4t m/s. Find the distance traveled from t = 1 to t = 3 seconds.

Show Solution

Distance = ∫₁³(6t² - 4t)dt = [2t³ - 2t²]₁³

= (2·27 - 2·9) - (2·1 - 2·1)

= (54 - 18) - (2 - 2)

= 36 - 0 = 36 meters

🌟 Challenge Problem

Advanced: Find the area between the curves y = x² and y = 2x from x = 0 to x = 2.

Show Solution

First, determine which function is on top. At x = 1: 2(1) = 2 > 1² = 1, so 2x is above x².

Area = ∫₀²(2x - x²)dx

= [x² - x³/3]₀²

= (4 - 8/3) - (0 - 0)

= 12/3 - 8/3 = 4/3 square units

🔄 Derivatives vs. Integrals

Derivative

  • Rate of change
  • Slope of tangent
  • Velocity from position
  • Makes functions smaller power

f(x) → f'(x)

Integral

  • Accumulation
  • Area under curve
  • Position from velocity
  • Makes functions bigger power

f(x) → ∫f(x)dx

🔑 Key Takeaways

  • Integration is the reverse of differentiation
  • Definite integrals calculate area under curves
  • Indefinite integrals find antiderivatives (don't forget +C!)
  • Power Rule for Integration: ∫xⁿdx = xⁿ⁺¹/(n+1) + C
  • Applications: total distance, area between curves, accumulated quantities
