Lesson 5: Factoring Quadratics

Breaking Down Quadratic Expressions

What is Factoring?

Factoring is the reverse of expanding. Instead of multiplying (x + 3)(x + 2) to get x² + 5x + 6, we start with x² + 5x + 6 and break it down into (x + 3)(x + 2).

Why Factor Quadratics?

  • 🎯 Find where a projectile hits the ground (roots/zeros)
  • 🎮 Determine when objects collide in games
  • 📐 Solve area and dimension problems
  • ⚡ Simplify complex expressions

Method 1: Factoring Simple Trinomials (a = 1)

For quadratics in the form x² + bx + c, find two numbers that:

  • Multiply to give c
  • Add to give b

Example 1: x² + 7x + 12

Step 1: Find two numbers that multiply to 12 and add to 7

Pairs that multiply to 12: (1,12), (2,6), (3,4)

Which pair adds to 7? → 3 and 4

Answer: (x + 3)(x + 4)

Check: (x + 3)(x + 4) = x² + 4x + 3x + 12 = x² + 7x + 12 ✓

Example 2: x² - 5x + 6

Step 1: Find two numbers that multiply to 6 and add to -5

Both numbers must be negative!

Pairs: (-1,-6), (-2,-3)

Which pair adds to -5? → -2 and -3

Answer: (x - 2)(x - 3)

Example 3: x² + 2x - 15

Step 1: Find two numbers that multiply to -15 and add to 2

One number is positive, one is negative

Pairs: (1,-15), (-1,15), (3,-5), (-3,5)

Which pair adds to 2? → 5 and -3

Answer: (x + 5)(x - 3)

Method 2: Factoring When a ≠ 1

For quadratics like 2x² + 7x + 3, we use the "ac method":

Example: 2x² + 7x + 3

Step 1: Multiply a × c = 2 × 3 = 6

Step 2: Find two numbers that multiply to 6 and add to 7

Numbers: 1 and 6

Step 3: Rewrite the middle term: 2x² + 1x + 6x + 3

Step 4: Factor by grouping:

= x(2x + 1) + 3(2x + 1)

= (x + 3)(2x + 1)

Example: 3x² - 10x + 8

Step 1: a × c = 3 × 8 = 24

Step 2: Find two numbers: multiply to 24, add to -10

Numbers: -4 and -6

Step 3: 3x² - 4x - 6x + 8

Step 4: = x(3x - 4) - 2(3x - 4)

= (x - 2)(3x - 4)

Special Factoring Patterns

Difference of Squares

Pattern: a² - b² = (a + b)(a - b)

Examples:

x² - 9 = (x + 3)(x - 3)

4x² - 25 = (2x + 5)(2x - 5)

x² - 16 = (x + 4)(x - 4)

Perfect Square Trinomials

Pattern: a² + 2ab + b² = (a + b)²

Examples:

x² + 6x + 9 = (x + 3)²

x² - 8x + 16 = (x - 4)²

4x² + 12x + 9 = (2x + 3)²

Real-World Application: Gaming Physics

🎮 Projectile Landing Problem

In a video game, a cannonball's height is given by:

h(t) = -5t² + 20t

When does the cannonball hit the ground (h = 0)?

Solution:

Set h(t) = 0: -5t² + 20t = 0

Factor out GCF: -5t(t - 4) = 0

Solutions: t = 0 (launch) or t = 4 seconds (landing)

Answer: The cannonball lands after 4 seconds

Practice Problems

Problem 1: Easy

Factor: x² + 9x + 14

Hint: Find two numbers that multiply to 14 and add to 9

Numbers: 2 and 7 (2 × 7 = 14, 2 + 7 = 9)

Answer: (x + 2)(x + 7)

Problem 2: Medium

Factor: x² - 3x - 18

Hint: One number is positive, one is negative. They multiply to -18 and add to -3

Numbers: -6 and 3 (-6 × 3 = -18, -6 + 3 = -3)

Answer: (x - 6)(x + 3)

Problem 3: Challenge

Factor: 2x² + 11x + 12

Hint: Use the ac method. a × c = 2 × 12 = 24

ac = 24, find numbers that add to 11: 3 and 8

2x² + 3x + 8x + 12

= x(2x + 3) + 4(2x + 3)

Answer: (x + 4)(2x + 3)

Problem 4: Special Pattern

Factor: x² - 49

Hint: This is a difference of squares

x² - 49 = x² - 7²

Answer: (x + 7)(x - 7)

Problem 5: Application

A basketball's height (in meters) is h(t) = -5t² + 15t. When does it hit the ground?

Hint: Set h(t) = 0 and factor

-5t² + 15t = 0

-5t(t - 3) = 0

t = 0 or t = 3

Answer: The ball hits the ground at 3 seconds

Key Takeaways

  • Factoring reverses multiplication to break expressions into factors
  • For x² + bx + c, find two numbers that multiply to c and add to b
  • Use the ac method when the leading coefficient isn't 1
  • Recognize special patterns: difference of squares and perfect squares
  • Factoring helps solve real-world problems like projectile motion
