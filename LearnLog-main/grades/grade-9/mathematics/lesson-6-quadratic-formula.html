Lesson 6: The Quadratic Formula

The Ultimate Tool for Solving Any Quadratic Equation

The Quadratic Formula

For ax² + bx + c = 0:

x = (-b ± √(b² - 4ac)) / 2a

This formula solves ANY quadratic equation, even when factoring is difficult or impossible!

Why the Quadratic Formula?

While factoring works great for simple quadratics, many equations can't be factored easily:

  • 🚫 x² + 3x + 1 = 0 (doesn't factor nicely)
  • 🚫 2x² - 5x - 2 = 0 (complex factoring)
  • ✅ The quadratic formula solves these instantly!

Understanding the Formula Components

For the equation ax² + bx + c = 0:

  • a = coefficient of x² term
  • b = coefficient of x term
  • c = constant term
  • ± means we get two solutions (one with +, one with -)
  • b² - 4ac is called the "discriminant" (tells us about solutions)

The Discriminant: Predicting Solutions

The discriminant Δ = b² - 4ac tells us what type of solutions to expect:

Discriminant Value Number of Solutions Type of Solutions Graph Behavior
Δ > 0 2 real solutions Two different numbers Parabola crosses x-axis twice
Δ = 0 1 real solution One repeated number Parabola touches x-axis once
Δ < 0 0 real solutions No real numbers work Parabola doesn't touch x-axis

Step-by-Step Examples

Example 1: x² + 5x + 6 = 0

Step 1: Identify a, b, c

a = 1, b = 5, c = 6

Step 2: Check discriminant

Δ = b² - 4ac = (5)² - 4(1)(6) = 25 - 24 = 1

Since Δ > 0, we have 2 real solutions

Step 3: Apply formula

x = (-5 ± √1) / 2(1)

x = (-5 ± 1) / 2

Step 4: Solve both cases

x = (-5 + 1) / 2 = -4/2 = -2

x = (-5 - 1) / 2 = -6/2 = -3

Solutions: x = -2 or x = -3

Note: This could also be factored as (x + 2)(x + 3) = 0

Example 2: 2x² - 7x + 3 = 0

Step 1: Identify a, b, c

a = 2, b = -7, c = 3

Step 2: Check discriminant

Δ = (-7)² - 4(2)(3) = 49 - 24 = 25

Δ > 0, so 2 real solutions

Step 3: Apply formula

x = (7 ± √25) / 2(2)

x = (7 ± 5) / 4

Step 4: Solve

x = (7 + 5) / 4 = 12/4 = 3

x = (7 - 5) / 4 = 2/4 = 0.5

Solutions: x = 3 or x = 0.5

Example 3: x² + 4x + 1 = 0 (Irrational Solutions)

Step 1: a = 1, b = 4, c = 1
Step 2: Δ = (4)² - 4(1)(1) = 16 - 4 = 12
Step 3: x = (-4 ± √12) / 2
Step 4: Simplify √12 = √(4 × 3) = 2√3

x = (-4 ± 2√3) / 2 = -2 ± √3

Solutions: x = -2 + √3 ≈ -0.27 or x = -2 - √3 ≈ -3.73

Real-World Application: Sports Physics

⚽ Soccer Ball Kick

A soccer ball is kicked with height equation:

h(t) = -4.9t² + 15t + 1

When does the ball hit the ground (h = 0)?

Solution

Equation: -4.9t² + 15t + 1 = 0

a = -4.9, b = 15, c = 1

Discriminant: Δ = (15)² - 4(-4.9)(1) = 225 + 19.6 = 244.6
Formula: t = (-15 ± √244.6) / 2(-4.9)

t = (-15 ± 15.64) / -9.8

Solutions:

t = (-15 + 15.64) / -9.8 = -0.065 (not physical, negative time)

t = (-15 - 15.64) / -9.8 = 3.13 seconds

Answer: The ball lands after approximately 3.13 seconds

Practice Problems

Problem 1: Basic

Solve: x² - 3x - 10 = 0

a = 1, b = -3, c = -10

Δ = (-3)² - 4(1)(-10) = 9 + 40 = 49

x = (3 ± √49) / 2 = (3 ± 7) / 2

x = 10/2 = 5 or x = -4/2 = -2

Answer: x = 5 or x = -2

Problem 2: With Leading Coefficient

Solve: 3x² + 8x - 3 = 0

a = 3, b = 8, c = -3

Δ = (8)² - 4(3)(-3) = 64 + 36 = 100

x = (-8 ± √100) / 6 = (-8 ± 10) / 6

x = 2/6 = 1/3 or x = -18/6 = -3

Answer: x = 1/3 or x = -3

Problem 3: One Solution

Solve: x² - 6x + 9 = 0

a = 1, b = -6, c = 9

Δ = (-6)² - 4(1)(9) = 36 - 36 = 0

One solution! x = (6 ± 0) / 2 = 3

Answer: x = 3 (double root)

Problem 4: Application

A ball's height is h(t) = -5t² + 20t + 2. When is it at ground level?

Set h(t) = 0: -5t² + 20t + 2 = 0

a = -5, b = 20, c = 2

Δ = 400 - 4(-5)(2) = 400 + 40 = 440

t = (-20 ± √440) / -10 = (-20 ± 20.98) / -10

t ≈ 4.10 seconds (positive solution)

Answer: Ball lands at about 4.1 seconds

Key Takeaways

  • The quadratic formula solves any quadratic equation: x = (-b ± √(b² - 4ac)) / 2a
  • The discriminant (b² - 4ac) predicts the number and type of solutions
  • Always check the discriminant before solving
  • Use ± to find both solutions (if they exist)
  • Real-world problems often produce one meaningful solution (positive time, positive distance, etc.)
