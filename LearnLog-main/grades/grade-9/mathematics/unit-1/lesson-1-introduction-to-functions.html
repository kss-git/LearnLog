What is a Function?

A function is a special relationship where each input has exactly one output. Think of it like your phone's screen brightness: when you set it to 50%, you get a specific brightness level every time.

Real-World Example: Social Media Likes

On Instagram, each post has a like count. The function could be:

f(post_id) = number_of_likes

Post #12345 might have 247 likes. Every time you check that post, you'll see the same number (until someone new likes it).

Function Notation

We write functions using notation like f(x), which reads as "f of x" or "the function f at x".

TikTok View Count Example

If a video gets 10 views per hour:

f(x) = 10x
where x = hours since posting

  • f(1) = 10 views after 1 hour
  • f(5) = 50 views after 5 hours
  • f(24) = 240 views after 1 day

App Algorithm Example: Spotify Recommendation

Spotify uses functions to recommend songs. Simplified version:

Recommendation Score Function

score(song) = (0.4 × genre_match) + (0.3 × popularity) + (0.3 × tempo_match)

Example for a song you might like:

  • Genre Match: 0.9 (you love pop)
  • Popularity: 0.7 (moderately popular)
  • Tempo Match: 0.8 (upbeat like you prefer)
  • Total Score: (0.4×0.9) + (0.3×0.7) + (0.3×0.8) = 0.81

Domain and Range

Domain: All possible inputs (x-values)
Range: All possible outputs (y-values or f(x) values)

YouTube Video Length Example

A YouTube video player shows time elapsed:

f(seconds_watched) = progress_percentage

For a 300-second (5-minute) video:

  • Domain: 0 to 300 seconds (you can't watch negative time or past the end)
  • Range: 0% to 100% (progress bar fills from empty to full)

Practice Problems

Problem 1: Discord Server Growth

A Discord server gains 15 new members each day. Write a function for total members after x days, starting with 50 members.

Show Solution

f(x) = 50 + 15x

Where x = days

After 10 days: f(10) = 50 + 15(10) = 200 members

Problem 2: Gaming XP System

In a game, you earn 100 XP per level completed. If g(n) represents total XP after n levels, find g(7).

Show Solution

g(n) = 100n

g(7) = 100 × 7 = 700 XP

Problem 3: Netflix Download Time

Downloading a movie takes 2 minutes per GB. Write a function for download time based on file size. What's the domain and range for movies between 1-5 GB?

Show Solution

t(g) = 2g (where g = gigabytes)

Domain: 1 ≤ g ≤ 5 (file sizes from 1 to 5 GB)

Range: 2 ≤ t ≤ 10 (download time from 2 to 10 minutes)

Key Takeaways

  • Functions map each input to exactly one output
  • Apps and algorithms use functions constantly (recommendations, scoring, timing)
  • Function notation: f(x) means "the output when input is x"
  • Domain = possible inputs, Range = possible outputs
  • Real-world functions often have practical limits on their domain and range
