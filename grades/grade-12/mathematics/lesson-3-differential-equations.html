Introduction to Differential Equations

Differential equations relate functions to their derivatives and are fundamental to modeling dynamic systems in physics, biology, economics, and engineering. Rather than finding values, we solve for entire functions that describe how systems change over time. This lesson covers first-order differential equations and their applications.

1. What is a Differential Equation?

A differential equation contains derivatives of an unknown function. The order is determined by the highest derivative present.

Examples

  • First-order: dy/dx = 2x (rate of change depends on x)
  • First-order: dy/dx = y (rate of change depends on y itself)
  • Second-order: d²y/dx² + y = 0 (harmonic oscillator)

Solution Types

  • General solution: Contains arbitrary constants (family of curves)
  • Particular solution: Satisfies initial conditions (specific curve)

2. Separable Differential Equations

A separable equation can be written as dy/dx = g(x)·h(y), allowing separation of variables.

Method

  1. Separate variables: (1/h(y)) dy = g(x) dx
  2. Integrate both sides: ∫(1/h(y)) dy = ∫g(x) dx
  3. Solve for y if possible

Example 1: Exponential Growth

Solve dy/dx = 0.5y with initial condition y(0) = 100

Solution:

Separate: (1/y) dy = 0.5 dx

Integrate: ∫(1/y) dy = ∫0.5 dx

ln|y| = 0.5x + C

y = Ae^(0.5x) where A = e^C

Apply y(0) = 100: 100 = Ae⁰ → A = 100

Particular solution: y = 100e^(0.5x)

Example 2: Newton's Law of Cooling

An object cools according to dT/dt = -k(T - T_ambient). If T(0) = 100°C, T_ambient = 20°C, and k = 0.1/min, find T(t).

Solution:

Rewrite: dT/dt = -0.1(T - 20)

Separate: dT/(T - 20) = -0.1 dt

Integrate: ln|T - 20| = -0.1t + C

T - 20 = Ae^(-0.1t)

Apply T(0) = 100: 80 = A

Solution: T(t) = 20 + 80e^(-0.1t)

As t → ∞, T → 20°C (ambient temperature)

3. Linear First-Order Equations

Standard form: dy/dx + P(x)y = Q(x)

Solution uses an integrating factor: μ(x) = e^(∫P(x)dx)

Method

  1. Write in standard form
  2. Find integrating factor μ(x) = e^(∫P(x)dx)
  3. Multiply both sides by μ(x)
  4. Left side becomes d/dx[μ(x)y]
  5. Integrate and solve for y

Example: Electric Circuit

An RL circuit with resistance R = 10Ω and inductance L = 2H has voltage V = 50V. Find current I(t) if I(0) = 0.

Differential equation: L(dI/dt) + RI = V → dI/dt + 5I = 25

Solution:

P(t) = 5, Q(t) = 25

Integrating factor: μ = e^(∫5 dt) = e^(5t)

Multiply: e^(5t)(dI/dt) + 5e^(5t)I = 25e^(5t)

d/dt[e^(5t)I] = 25e^(5t)

Integrate: e^(5t)I = 5e^(5t) + C

I = 5 + Ce^(-5t)

Apply I(0) = 0: C = -5

Solution: I(t) = 5(1 - e^(-5t)) amperes

4. Population Models

Exponential Growth (Malthusian Model)

dP/dt = kP (growth rate proportional to population)

Solution: P(t) = P₀e^(kt)

Logistic Growth Model

Accounts for carrying capacity K (environmental limits):

dP/dt = kP(1 - P/K)

Solution (using separation of variables and partial fractions):

P(t) = K/(1 + Ae^(-kt)) where A = (K - P₀)/P₀

Example: Bacteria Culture

A bacteria population grows logistically with k = 0.8/hour and K = 10,000. If P(0) = 1,000, find P(t).

Solution:

A = (10,000 - 1,000)/1,000 = 9

P(t) = 10,000/(1 + 9e^(-0.8t))

At t = 5 hours: P(5) = 10,000/(1 + 9e^(-4)) ≈ 9,950 bacteria

Population approaches carrying capacity of 10,000

5. Mixing Problems

These involve tracking concentration of substances in tanks or containers.

Example: Salt Tank

A 100-liter tank initially contains 20 kg of salt. Pure water flows in at 5 L/min and the mixture flows out at 5 L/min. Find the salt amount S(t).

Solution:

Rate of change: dS/dt = (rate in) - (rate out)

Rate in = 0 kg/min (pure water)

Rate out = (S/100) kg/L × 5 L/min = S/20 kg/min

dS/dt = -S/20

This is separable: dS/S = -dt/20

ln|S| = -t/20 + C

S = Ae^(-t/20)

Apply S(0) = 20: A = 20

Solution: S(t) = 20e^(-t/20) kg

After 20 minutes: S(20) = 20e^(-1) ≈ 7.36 kg

6. Applications in Economics

Continuous Compound Interest

If money grows continuously at rate r: dA/dt = rA

Solution: A(t) = A₀e^(rt)

Supply and Demand Dynamics

Price adjustment based on excess demand:

dP/dt = k(D(P) - S(P))

Example: Investment Growth

$10,000 invested at 6% annual rate compounded continuously. Find value after 10 years.

Solution:

dA/dt = 0.06A with A(0) = 10,000

A(t) = 10,000e^(0.06t)

A(10) = 10,000e^(0.6) ≈ $18,221

Key Takeaways

  • Differential equations model dynamic systems where rates of change are known
  • Separable equations: rewrite as f(y)dy = g(x)dx and integrate both sides
  • Linear first-order: use integrating factor μ(x) = e^(∫P(x)dx)
  • Exponential growth: unlimited increase, dP/dt = kP
  • Logistic growth: approaches carrying capacity, dP/dt = kP(1 - P/K)
  • Applications span physics (cooling, circuits), biology (populations), economics (interest, price dynamics)
  • Initial conditions determine the particular solution from the general solution family
