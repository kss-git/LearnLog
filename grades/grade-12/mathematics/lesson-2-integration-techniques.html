Introduction to Advanced Integration

While basic integration reverses differentiation, advanced techniques allow us to integrate complex functions, solve area and volume problems, and model accumulation in physics and economics. This lesson covers integration by parts, partial fractions, trigonometric substitution, and numerical methods.

1. Integration by Parts

Based on the product rule, integration by parts is used when integrating products of functions:

Formula: ∫u dv = uv - ∫v du

Strategy: LIATE Rule

Choose u in this priority order:

  • Logarithmic functions: ln(x), log(x)
  • Inverse trig functions: arcsin(x), arctan(x)
  • Algebraic functions: x², x + 1
  • Trigonometric functions: sin(x), cos(x)
  • Exponential functions: eˣ, 2ˣ

Example 1: ∫x·eˣ dx

Solution:

Let u = x (algebraic), dv = eˣ dx

Then du = dx, v = eˣ

∫x·eˣ dx = x·eˣ - ∫eˣ dx = x·eˣ - eˣ + C = eˣ(x - 1) + C

Example 2: ∫ln(x) dx

Solution:

Let u = ln(x), dv = dx

Then du = (1/x)dx, v = x

∫ln(x) dx = x·ln(x) - ∫x·(1/x) dx = x·ln(x) - x + C

Application: Electric Circuit

The charge Q(t) stored in a capacitor in an RC circuit with voltage V(t) = te⁻ᵗ satisfies:

Q(t) = ∫te⁻ᵗ dt

Using integration by parts:

Q(t) = -te⁻ᵗ - e⁻ᵗ + C = -(t + 1)e⁻ᵗ + C

2. Partial Fraction Decomposition

Used to integrate rational functions by breaking them into simpler fractions.

Example: ∫(3x + 5)/((x + 1)(x + 2)) dx

Step 1: Decompose into partial fractions

(3x + 5)/((x + 1)(x + 2)) = A/(x + 1) + B/(x + 2)

3x + 5 = A(x + 2) + B(x + 1)

Setting x = -1: -3 + 5 = A(1) → A = 2

Setting x = -2: -6 + 5 = B(-1) → B = 1

Step 2: Integrate

∫[2/(x + 1) + 1/(x + 2)] dx = 2ln|x + 1| + ln|x + 2| + C

Application: Population Growth Model

A logistic growth model with carrying capacity K:

dP/dt = kP(1 - P/K)

Separating variables and using partial fractions:

∫dP/(P(K - P)) = ∫(k/K) dt

Using partial fractions: 1/(P(K - P)) = 1/(KP) + 1/(K(K - P))

Solution: P(t) = K/(1 + Ae⁻ᵏᵗ)

3. Trigonometric Substitution

Used for integrals involving √(a² - x²), √(a² + x²), or √(x² - a²).

Substitution Guide

  • √(a² - x²): Use x = a·sin(θ)
  • √(a² + x²): Use x = a·tan(θ)
  • √(x² - a²): Use x = a·sec(θ)

Example: ∫√(4 - x²) dx

Solution:

Let x = 2sin(θ), dx = 2cos(θ)dθ

√(4 - x²) = √(4 - 4sin²θ) = 2cos(θ)

∫√(4 - x²) dx = ∫2cos(θ)·2cos(θ) dθ = 4∫cos²(θ) dθ

Using cos²(θ) = (1 + cos(2θ))/2:

= 2∫(1 + cos(2θ)) dθ = 2θ + sin(2θ) + C

= 2arcsin(x/2) + (x/2)√(4 - x²) + C

Application: Circular Motion

Finding the area under a semicircle of radius 2:

Area = ∫₋₂² √(4 - x²) dx = 2π (half the area of circle with radius 2)

4. Numerical Integration

When analytical integration is difficult or impossible, numerical methods approximate definite integrals.

Trapezoidal Rule

∫ₐᵇ f(x) dx ≈ (Δx/2)[f(x₀) + 2f(x₁) + 2f(x₂) + ... + 2f(xₙ₋₁) + f(xₙ)]

where Δx = (b - a)/n

Simpson's Rule

More accurate, using parabolic approximations:

∫ₐᵇ f(x) dx ≈ (Δx/3)[f(x₀) + 4f(x₁) + 2f(x₂) + 4f(x₃) + ... + f(xₙ)]

Example: Estimating ∫₀¹ e^(-x²) dx

This integral has no elementary antiderivative, so numerical methods are essential.

Using Trapezoidal Rule with n = 4:

Δx = 1/4 = 0.25

Points: x₀ = 0, x₁ = 0.25, x₂ = 0.5, x₃ = 0.75, x₄ = 1

f(x₀) = 1, f(x₁) ≈ 0.9394, f(x₂) ≈ 0.7788, f(x₃) ≈ 0.5698, f(x₄) ≈ 0.3679

Approximation: (0.25/2)[1 + 2(0.9394) + 2(0.7788) + 2(0.5698) + 0.3679] ≈ 0.7430

Actual value ≈ 0.7468 (error about 0.5%)

5. Applications in Physics and Engineering

Work Done by Variable Force

Work = ∫ₐᵇ F(x) dx

Example: Spring Compression

A spring with constant k = 500 N/m is compressed from x = 0 to x = 0.2 m. Find the work done.

Solution:

Hooke's Law: F(x) = kx = 500x

Work = ∫₀^0.2 500x dx = 250x²|₀^0.2 = 250(0.04) = 10 Joules

Center of Mass

For a continuous distribution with density ρ(x):

x̄ = ∫x·ρ(x) dx / ∫ρ(x) dx

6. Economic Applications

Consumer Surplus

The difference between what consumers are willing to pay and what they actually pay.

If demand function is p = D(q) and equilibrium is (Q*, P*):

Consumer Surplus = ∫₀^Q* D(q) dq - P*·Q*

Practice Problems

Problem 1: Integration by Parts

Evaluate ∫x²·sin(x) dx

Problem 2: Partial Fractions

Evaluate ∫(2x + 3)/((x - 1)(x + 2)) dx

Problem 3: Trigonometric Substitution

Evaluate ∫₀² x²√(4 - x²) dx

Problem 4: Applied Problem

A force F(x) = 2x + 3 N moves an object from x = 1 m to x = 5 m. Calculate the work done.

Problem 5: Numerical Integration

Use the Trapezoidal Rule with n = 4 to approximate ∫₁² (1/x) dx and compare with the exact value ln(2).

Key Takeaways

  • Integration by parts reverses the product rule: ∫u dv = uv - ∫v du
  • Partial fractions decompose rational functions into simpler integrable terms
  • Trigonometric substitution handles radicals like √(a² - x²)
  • Numerical methods (Trapezoidal, Simpson's) approximate integrals when exact solutions are impossible
  • Applications include work, area, volume, center of mass, and economic surplus
  • Choose the right technique based on the integrand's form
