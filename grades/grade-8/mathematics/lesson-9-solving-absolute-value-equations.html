⚽ Imagine you kick a ball into the air. It goes up, reaches a peak, then comes down. That curved path is described by a quadratic equation. In this lesson, we learn how to find the exact points where these equations equal zero!

🎯 Learning Objectives

  • Solve quadratic equations by factoring
  • Use the zero-product property
  • Find all solutions of quadratic equations
  • Check answers by substitution

📚 What is a Quadratic Equation?

A quadratic equation is an equation in the form:

ax² + bx + c = 0

Where a ≠ 0 and the highest power of x is 2.

💡 Key Idea: Zero Product Property

If:

(x + a)(x + b) = 0

Then:

x + a = 0 OR x + b = 0

This is the key to solving quadratic equations!

🧠 Example 1

Solve:

x² + 5x + 6 = 0

Factor:

(x + 2)(x + 3) = 0

Solutions:

x + 2 = 0 → x = -2

x + 3 = 0 → x = -3

Final Answer: x = -2, -3

🧠 Example 2

Solve:

x² - x - 12 = 0

Factor:

(x - 4)(x + 3) = 0

Solutions:

x - 4 = 0 → x = 4

x + 3 = 0 → x = -3

Final Answer: x = 4, -3

🎯 Practice Problem

Solve:

x² + 7x + 10 = 0

📌 Summary

  • Quadratic equations are in form ax² + bx + c = 0
  • Always try factoring first
  • Use zero-product property
  • Set each factor equal to zero
