🎯 Imagine you're throwing a dart. It lands 5 units away from the center — it could be +5 or -5. Absolute value measures distance from zero, not direction. That’s why we get TWO answers in these equations!

🎯 Learning Objectives

  • Understand absolute value as distance from zero
  • Solve absolute value equations
  • Recognize two solutions from one equation
  • Check solutions by substitution

📚 What is Absolute Value?

Absolute value shows distance from zero.

|x| = distance from 0

|5| = 5

|-5| = 5

💡 Key Idea

|x| = a

x = a OR x = -a

🧠 Example 1

|x| = 7

x = 7, x = -7

🧠 Example 2

|x - 3| = 5

x - 3 = 5 → x = 8

x - 3 = -5 → x = -2

Answer: x = 8, -2

🌍 Real-World Example

  • GPS distance
  • Temperature differences
  • Game score differences
  • Engineering tolerances

🎯 Practice Problem

|x + 4| = 9

📌 Summary

  • Absolute value = distance from zero
  • |x| always positive
  • Two solutions: x = a or x = -a
