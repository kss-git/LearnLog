📘 What is an Absolute Value Equation?

An absolute value equation is an equation that contains an absolute value expression.

|x| = a

This means the distance from zero is always positive.

🧠 Rule

If:

|x| = a

Then:

x = a or x = -a

🧠 Example 1

Solve:

|x| = 5

Solution:

x = 5 or x = -5

🧠 Example 2

Solve:

|x - 3| = 4

Split into two cases:

x - 3 = 4 OR x - 3 = -4

Solutions:

x = 7 or x = -1

🎯 Practice Problem

Solve:

|x + 2| = 6
