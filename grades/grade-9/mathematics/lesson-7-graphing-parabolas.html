Lesson 7: Graphing Parabolas

Visualizing Quadratic Functions

What is a Parabola?

A parabola is the U-shaped curve created by graphing a quadratic function

y = ax² + bx + c

Basic Parabola: y = x²

The simplest parabola opens upward with vertex at the origin

Key Features of a Parabola

🎯 Vertex

The highest or lowest point on the parabola

Formula: x = -b/2a

Substitute to find y-coordinate

📏 Axis of Symmetry

Vertical line through the vertex

Equation: x = -b/2a

Parabola mirrors on both sides

↕️ Direction

Determined by coefficient a:

a > 0: Opens upward ∪

a < 0: Opens downward ∩

📍 Y-Intercept

Where parabola crosses y-axis

Point: (0, c)

Found by setting x = 0

🎪 X-Intercepts (Roots)

Where parabola crosses x-axis

Found by solving ax² + bx + c = 0

Can be 0, 1, or 2 intercepts

📐 Width

Determined by |a|:

|a| > 1: Narrow

|a| < 1: Wide

Effect of Coefficient 'a'

Comparing Different Values of 'a'

Standard Form vs. Vertex Form

Form Equation Best For Shows Directly
Standard Form y = ax² + bx + c Finding y-intercept Y-intercept: (0, c)
Vertex Form y = a(x - h)² + k Finding vertex, graphing Vertex: (h, k)

Step-by-Step Graphing Examples

Example 1: y = x² - 4x + 3

Step 1: Find the vertex

x = -b/2a = -(-4)/2(1) = 4/2 = 2

y = (2)² - 4(2) + 3 = 4 - 8 + 3 = -1

Vertex: (2, -1)

Step 2: Find y-intercept (set x = 0)

y = 0² - 4(0) + 3 = 3

Y-intercept: (0, 3)

Step 3: Find x-intercepts (solve y = 0)

x² - 4x + 3 = 0

(x - 1)(x - 3) = 0

X-intercepts: (1, 0) and (3, 0)

Step 4: Note that a = 1 > 0, so parabola opens upward

Example 2: y = -2x² + 4x + 1

Step 1: Vertex

x = -4/2(-2) = -4/-4 = 1

y = -2(1)² + 4(1) + 1 = -2 + 4 + 1 = 3

Vertex: (1, 3)

Step 2: Y-intercept: (0, 1)

Step 3: Direction: a = -2 < 0, opens downward

Step 4: Width: |a| = 2 > 1, narrower than y = x²

Real-World Application: Basketball Shot

🏀 Modeling a Basketball Trajectory

Height equation: h(d) = -0.1d² + 0.8d + 2

where d is horizontal distance in meters

Analysis

Initial Height (release point): h(0) = 2 meters

Maximum Height (vertex):

d = -0.8/2(-0.1) = -0.8/-0.2 = 4 meters

h(4) = -0.1(16) + 0.8(4) + 2 = -1.6 + 3.2 + 2 = 3.6 meters

Peak is at (4, 3.6)

Practice Problems

Problem 1: Find Key Features

For y = x² - 6x + 8, find the vertex, axis of symmetry, and intercepts.

Vertex: x = -(-6)/2(1) = 3, y = 9 - 18 + 8 = -1 → (3, -1)

Axis of symmetry: x = 3

Y-intercept: (0, 8)

X-intercepts: (x - 2)(x - 4) = 0 → (2, 0) and (4, 0)

Problem 2: Direction and Width

Does y = -0.5x² + 2x - 1 open up or down? Is it wider or narrower than y = x²?

Direction: a = -0.5 < 0, opens downward

Width: |a| = 0.5 < 1, wider than y = x²

Problem 3: Application

A fountain's water height is h(d) = -d² + 4d, where d is horizontal distance. What's the maximum height?

Vertex: d = -4/2(-1) = 2

h(2) = -(2)² + 4(2) = -4 + 8 = 4

Maximum height: 4 meters at d = 2 meters

Key Takeaways

  • Parabolas are U-shaped curves from quadratic functions
  • The vertex is found at x = -b/2a
  • Coefficient 'a' determines direction (up/down) and width
  • Y-intercept is always (0, c)
  • X-intercepts are found by solving ax² + bx + c = 0
  • Parabolas model many real-world phenomena: projectiles, arches, satellite dishes
