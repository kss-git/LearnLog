What is a Limit?

A limit describes what value a function approaches as the input gets closer and closer to a certain point. It's like asking: "Where is this function heading?"

We write: limx→a f(x) = L

This means: "As x gets closer to a, f(x) gets closer to L"

📊 Visualizing Limits

Consider the function f(x) = (x² - 1)/(x - 1). What happens as x approaches 1?

Notice: Even though f(1) is undefined (division by zero!), the limit as x approaches 1 is 2. We can simplify: (x² - 1)/(x - 1) = (x+1)(x-1)/(x-1) = x + 1, which equals 2 when x = 1.

🚗 Real-World Example: Velocity

Imagine a car traveling along a straight road. Its position at time t seconds is given by s(t) = 5t² meters.

Question: What is the car's velocity at exactly t = 2 seconds?

Solution using limits:

Velocity = rate of change of position = limh→0 [s(2+h) - s(2)]/h

= limh→0 [5(2+h)² - 5(2)²]/h

= limh→0 [5(4 + 4h + h²) - 20]/h

= limh→0 [20 + 20h + 5h² - 20]/h

= limh→0 [20h + 5h²]/h

= limh→0 (20 + 5h) = 20 m/s

📝 Types of Limits

1. Two-Sided Limit

limx→a f(x) exists when the function approaches the same value from both left and right.

2. One-Sided Limits

limx→a⁺ f(x) (from the right) and limx→a⁻ f(x) (from the left)

3. Infinite Limits

limx→a f(x) = ∞ when the function grows without bound.

🎯 Practice Problems

Problem 1: Basic Limit

Find: limx→3 (x² + 2x - 1)

Show Solution

Simply substitute x = 3:

(3)² + 2(3) - 1 = 9 + 6 - 1 = 14

Problem 2: Limit with Factoring

Find: limx→2 (x² - 4)/(x - 2)

Show Solution

Factor the numerator:

(x² - 4)/(x - 2) = (x + 2)(x - 2)/(x - 2) = x + 2

Now substitute: 2 + 2 = 4

Problem 3: One-Sided Limit

For f(x) = |x|/x, find: limx→0⁺ f(x) and limx→0⁻ f(x)

Show Solution

From the right (x > 0): |x|/x = x/x = 1

From the left (x < 0): |x|/x = -x/x = -1

Since the limits differ, limx→0 f(x) does not exist!

🌟 Challenge Problem

Advanced: A ball is thrown upward from the ground with initial velocity 30 m/s. Its height is h(t) = 30t - 5t² meters. Use limits to find the instantaneous velocity at t = 3 seconds.

Show Solution

v(3) = limh→0 [h(3+h) - h(3)]/h

= limh→0 [30(3+h) - 5(3+h)² - (90 - 45)]/h

= limh→0 [90 + 30h - 5(9 + 6h + h²) - 45]/h

= limh→0 [90 + 30h - 45 - 30h - 5h² - 45]/h

= limh→0 [-5h²]/h = limh→0 -5h = 0 m/s

The ball has stopped momentarily at its maximum height!

🔑 Key Takeaways

  • Limits describe the behavior of functions as we approach a point
  • A limit can exist even when the function value doesn't
  • Limits are the foundation for derivatives and integrals
  • We use limits to find instantaneous rates of change (like velocity)
